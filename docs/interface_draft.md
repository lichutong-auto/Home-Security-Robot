# 视觉与底盘通信接口草案 (v0.1)
> 目标：定义视觉模块如何向底盘驱动模块发送指令，实现松耦合开发。

## 通信方式
- **物理**：串口 (UART)
- **波特率**：115200 (或9600，可协商)
- **数据格式**：字符串指令，以换行符 `\n` 结束。

## 指令集草案（例）
视觉模块发送 -> 底盘模块接收并执行：
- `FORWARD,200`  # 前进，速度值200
- `BACKWARD,150` # 后退，速度值150
- `TURN_LEFT,90` # 左转90度
- `STOP`         # 停止
- `SENSOR,25.6,142` # 传感器数据（温度,距离），用于调试显示
